# Recession Prediction ML

This project uses machine learning to classify economic conditions as either recession or 
no recession based on macroeconomic indicators. The goal is to build an end-to-end
classification pipeline that includes data preprocessing, feature selection, model training, 
and performance evaluation.

## Project Overview

The project compares different supervised machine learning models to predict recession status from economic data.
The steps invovle cleaning the dataset, preparing features, splitting the data in a time-aware way, training 
classification models, and evaluating performance using metrics such as accuracy, recall, F1-score, and 
confusion matrices.

## Models Used

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors

## Skills Demonstrated

- Data preprocessing
- Feature selection
- Time-aware train/test splitting
- Supervised machine learning
- Classification model evaluation
- Confusion matrix analysis
- Python, pandas, scikit-learn, matplotlib
